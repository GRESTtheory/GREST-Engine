# GREST-Engine
**Geometric Resistance of Expanding Space-Time (GREST): Galactic Dynamics Validator**

## Overview
This repository contains the Python implementation of the GREST framework. It validates the theory across three distinct astrophysical scales: Dwarf Stars, Spiral Galaxies, and Massive Super-Spirals.

**Paper Preprint:** https://doi.org/10.5281/zenodo.18473062
**DOI:** 10.5281/zenodo.18473062

## Validation Suite
1. **grest_engine.py**: The core theoretical model.
2. **grest_real_test.py**: Validation against NGC 6503 (Dwarf Spiral).
3. **grest_test_universal.py**: Universality Test (NGC 6503 vs NGC 7331).
4. **grest_test_binaries.py**: **Micro-Scale Test** (Wide Binaries/Gaia Data) - The "Holy Grail" low-acceleration limit.
